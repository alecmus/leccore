Detailed Description

Application settings base class. This class is for saving/loading application settings.

Member Function Documentation

◆ base()

settings & liblec::leccore::settings::base ( )

Get a reference to the base class.

Returns
A reference to the base class.

Useful for use in the ternary operator, e.g. if r is and instance of registry_settings and i is an instance of inisettings then one can initialize a settings reference s by calling s = condition ? r : i.
