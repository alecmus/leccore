Detailed Description

Settings derived class that uses ini files. For saving/loading application settings using ini configuration files.

Get the currently set ini path.

Parameters
ini_pathThe path to the folder the ini file is to be stored in.
errorError information.
Returns
Returns true if successful, else false.

If the set_ini_path method is not called then the ini path is set internally to "C:\Users\user-name\AppData\Local\company-name\app-name" as read from the application's version information. The app_version_info class is called internally for this. If the current application does not have any version information then this method will return an error.

void liblec::leccore::ini_settings::set_ini_path ( const std::string &  ini_path)

Set the folder under which the ini file is to be saved, e.g. "C:\Users\user-name\AppData\Local\com.github.alecmus\PC Info".

Parameters
ini_pathThe path to the folder the ini file is to be saved in. Use an empty string to store the file in the same directory as the application (useful when running the application in portable-mode).

If this method is not called the ini path is set internally to "C:\Users\user-name\AppData\Local\company-name\app-name" as read from the application's version information. The app_version_info class is called internally for this. If the current application does not have any version information then this method will return an error.
