enum class liblec::leccore::zip::compression_level

The compression level to use.

Enumerator
normal 

Balance between size and speed. The default.

maximum 

For the smallest archive size, but takes longer.

fast 

Compromises compression level in favor of speed.

superfast 

Lowest compression level for super fast zipping.

none 

No compression. Use this for the ultimate speed when you don't need any compression and just want to store files.

Start zipping operation.

Parameters
filenameThe target filename, including the extension.
entriesThe archive entries (files, directories).
levelThe compression level, as defined in the compression_level enumeration.

This method returns almost immediately. The actual zipping is executed on a seperate thread. To check the status of the zipping call the zipping method.

bool liblec::leccore::zip::zipping ( )

Check whether the zipping operation is still underway.

Returns
Returns true if the zipping is still underway, else false.

After calling start call this method in a loop or a timer, depending on your kind of app, then call result once it returns false.
