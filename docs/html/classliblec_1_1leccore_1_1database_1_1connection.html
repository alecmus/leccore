Execute an sql statement.

Parameters
sqlThe query (supports placeholders for binding), e.g. INSERT INTO Members VALUES(?, ?, ?, ?);
valuesThe values to bind to the sql statement, e.g. { "John", "Doe", 172, blob{ jpg_file_data } }
errorError information.
Returns
Returns true if successful, else false.

The use of placeholders is optional. You can make the sql statement yourself and leave the values parameter empty but this is not recommended because of sql injection attacks and issues like the proper handling of special characters, which quickly becomes tedious and is error prone.
