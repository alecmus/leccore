Get the currently set registry path.

Parameters
registry_pathThe registry path, relative to the scope defined in the constructor.
errorError information.
Returns
Returns true if successful, else false.

If set_registry_path is not called then the registry path is internally set to "Software\\company-name\\app-name" as read from the application's version information. The app_version_info class is called internally for this. If the current application does not have any version information then this method will return an error.
