◆ hash_results

using liblec::leccore::hash_file::hash_results = std::map<algorithm, std::string>

Hash results. Key is the algorithm and value is the hash.

Allows running multiple hashes in one sweep, which is far much more efficient than running one hash at a time because in that case the file data has to be read again.

bool liblec::leccore::hash_file::hashing ( )

Check whether hashing is still in progress.

Returns
Returns true if the hashing process is still underway and false otherwise.

After calling start call this method in a loop or a timer, depending on your kind of app, then call result once it returns false.
