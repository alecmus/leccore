Enumerator
current_user 

Settings for the currently logged in user only.

Settings for everyone that uses this computer.

bool liblec::leccore::registry::do_delete ( const std::string &  path,

)

Delete a sub-key if it's empty.

Parameters
pathThe path, relative to the scope as defined in the scope enumeration, e.g. Software\com.github.alecmus\PC Info
errorError information.
Returns
True if successful, else false.

Only deletes the sub-key if it's empty. If the sub-key is not empty an access is denied error is returned. This is a useful method to call on the Software\company-name sub-key after deleting a specific application's sub-key, e.g. after deleting the sub-key Software\com.github.alecmus\PC Info using the do_recursive_delete it's a great idea to then call this method. If no settings for other applications are stored under the Software\com.github.alecmus subkey then it is deleted as well, leaving a clean registry. This is highly recommended as it leaves a clean registry after all apps for a given company are uninstalled.
