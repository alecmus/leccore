◆ checking()

bool liblec::leccore::check_update::checking ( )

Check whether updates are being checked for [sic] :)

Returns
Returns true if the update checking process is still underway and false otherwise.

After calling start call this method in a loop or a timer, depending on your kind of app, then call result once it returns false.

void liblec::leccore::check_update::start ( )

Start checking for updates.

This method returns almost immediately. The actual checking is executed on a different thread. To check the status of the checking use the checking method.
