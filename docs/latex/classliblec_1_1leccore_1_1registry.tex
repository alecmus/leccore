Only deletes the sub-\/key if it\textquotesingle{}s empty. If the sub-\/key is not empty an access is denied error is returned. This is a useful method to call on the Software\textbackslash{}company-\/name sub-\/key after deleting a specific application\textquotesingle{}s sub-\/key, e.\+g. after deleting the sub-\/key Software\textbackslash{}com.\+github.\+alecmus\textbackslash{}\+PC Info using the \mbox{\hyperlink{classliblec_1_1leccore_1_1registry_a6f5a1a609c609f209e6270caf7c7c484}{do\+\_\+recursive\+\_\+delete}} it\textquotesingle{}s a great idea to then call this method. If no settings for other applications are stored under the Software\textbackslash{}com.\+github.\+alecmus subkey then it is deleted as well, leaving a clean registry.
